# Library and Projects

Iris separates project browsing from active editing.

## Projects page

The projects route is the saved-work surface for the product.

It shows:

- the user's existing videos
- counts and project metadata
- a direct path into a new reel or an existing edit

## Library behavior inside the editor

Inside the editor, the library is the local media pool for all media assets associated with the current project, including uploaded source clips and generated variants.

When the editor is in 'vibe' mode, the Library sidebar is hidden. To streamline the workflow, any new media uploaded in 'vibe' mode is automatically added to the active timeline, as the usual 'add to timeline' affordance from the library is not visible.
