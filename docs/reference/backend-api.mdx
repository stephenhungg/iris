# Backend API

This page is a practical route reference, not a formal OpenAPI spec. All routes are prefixed with `/api`.

## Health and status

### `GET /api/health`

Returns server liveness status.

### `GET /api/me`

Returns the currently authenticated user profile.

### `GET /api/ai/health`

Returns AI provider reachability and configuration status.

### `GET /api/ai/timeline`

Returns a timeline of recent AI operations with timing information.

## Media Previews

These endpoints provide various ways to preview media from a project, supporting timeline scrubbing, filmstrip generation, and range playback.

### `GET /api/preview/{id}/frame`

Returns a single preview JPEG image for a specific timestamp within a project.

**Path Parameters:**
- `id`: The ID of the project or media asset.

**Query Parameters:**
- `ts` (float, required): The timeline-space timestamp in seconds for which to retrieve the frame. Must be non-negative.

**Response:**
```json
{
  "ts": 10.5,
  "url": "https://example.com/path/to/frame_10_5.jpg"
}
```

### `GET /api/preview/{id}/strip`

Returns an ordered filmstrip of preview frames covering a specified time range.

**Path Parameters:**
- `id`: The ID of the project or media asset.

**Query Parameters:**
- `start` (float, required): The start timestamp (in seconds) of the range.
- `end` (float, required): The end timestamp (in seconds) of the range.
- `fps` (float, required): The desired frames per second for the strip.

**Response:**
```json
{
  "frames": [
    {
      "ts": 0.0,
      "url": "https://example.com/path/to/frame_0.jpg"
    },
    {
      "ts": 1.0,
      "url": "https://example.com/path/to/frame_1.jpg"
    }
  ]
}
```

### `GET /api/preview/{id}/range`

Returns a URL to a rendered low-resolution MP4 video covering a specified slice of the timeline.

**Path Parameters:**
- `id`: The ID of the project or media asset.

**Query Parameters:**
- `start` (float, required): The start timestamp (in seconds) of the range.
- `end` (float, required): The end timestamp (in seconds) of the range.

**Response:**
```json
{
  "preview_url": "https://example.com/path/to/preview_range.mp4",
  "duration": 5.0
}
```
