## Commands

### `iris init`

Initializes a new Iris project.

```bash
iris init <project_name>
```

### `iris upload`

Uploads a video file to the Iris library.

```bash
iris upload --file <path_to_video> --project <project_id>
```

### `iris generate`

Generates a new video variant based on a prompt.

```bash
iris generate --segment <segment_id> --prompt "make this brighter"
```

### `iris accept`

Accepts a generated variant, applying it to the timeline.

```bash
iris accept --variant <variant_id>
```

### `iris narrate`

Generates a voiceover for a segment using a given prompt.

```bash
iris narrate --segment <segment_id> --prompt "a calm voice describes the scene"
```

### `iris preview`

Previews frames, strips, or ranges from a project.

- **Preview a single frame:**
  ```bash
  iris preview <project_id> --frame 10.5
  ```
- **Preview a frame strip:**
  ```bash
  iris preview <project_id> --strip 5.0 15.0 --fps 2.0
  ```
- **Preview a time range:**
  ```bash
  iris preview <project_id> --range 0.0 30.0
  ```

### `iris batch-generate`

Submits a batch generation request from a JSON file.

```bash
iris batch-generate --edits ./path/to/edits.json
```

### `iris batch-accept`

Submits a batch accept request from a JSON file.

```bash
iris batch-accept --accepts ./path/to/accepts.json
```

### `iris grade`

Applies or previews color grading adjustments for a segment.

- **Apply brightness adjustment:**
  ```bash
  iris grade --segment <segment_id> --brightness 0.1
  ```
- **Preview saturation adjustment:**
  ```bash
  iris grade --segment <segment_id> --saturation 0.2 --preview
  ```

### `iris remix`

Creates a remix from an existing variant using a modifier prompt.

- **Remix with composition preserved:**
  ```bash
  iris remix --variant <variant_id> --modifier "add a vintage film look"
  ```
- **Remix without preserving composition:**
  ```bash
  iris remix --variant <variant_id> --modifier "change the background entirely" --no-preserve-composition
  ```

### `iris score`

Scores variants or project continuity.

- **Score a single variant:**
  ```bash
  iris score --variant <variant_id> --compare-to "prompt"
  ```
- **Compare multiple variants:**
  ```bash
  iris score --compare <variant_id_1> <variant_id_2>
  ```
- **Score project continuity:**
  ```bash
  iris score --continuity <project_id>
  ```

### `iris snapshot`

Creates a snapshot of the current project timeline.

```bash
iris snapshot <project_id>
```

### `iris revert`

Reverts the project timeline to a specified snapshot.

```bash
iris revert <project_id> --snapshot <snapshot_id>
```
