## Installation and Agent Setup

To install the Iris CLI and set up the agent skill for prompt-driven video editing, follow these steps:

1.  **Install the Iris CLI:**
    ```bash
    pip install iris-edit
    ```
2.  **Set up the agent skill file:** This makes the Iris CLI available to AI agents (e.g., Claude).
    ```bash
    mkdir -p ~/.claude/skills/iris-edit && curl -sL https://raw.githubusercontent.com/stephenhungg/iris/main/cli/SKILL.md -o ~/.claude/skills/iris-edit/SKILL.md
    ```
3.  **Authenticate with Iris:**
    ```bash
    iris auth login --base-url https://useiris.tech
    ```

Once installed and authenticated, you can start editing videos by saying "iris edit" to your agent.
