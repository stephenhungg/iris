## Installation and Authentication

To get started with the Iris CLI, install it from the local repository and authenticate it against your Iris backend.

```bash
pip install -e ./cli
iris auth login --base-url http://localhost:8000
```

The `iris auth login` command will prompt you for your credentials and store an authentication token for subsequent CLI commands.

## Agent Skill Integration

Iris provides a pre-packaged agent skill that allows AI agents (like Claude, Codex, or Cursor) to interact with the Iris CLI for automated video editing tasks.

### For Claude Code User

You can install the Iris agent skill directly into your Claude environment with a single command:

```bash
mkdir -p ~/.claude/skills/iris-edit && curl -sL https://raw.githubusercontent.com/stephenhungg/iris/main/cli/SKILL.md -o ~/.claude/skills/iris-edit/SKILL.md
```

### For Other Agents

For agents like Codex, Openclaw, or Cursor, you can integrate the Iris skill by pasting the contents of the `cli/SKILL.md` file directly into your agent's system prompt or configuration. This provides the agent with the necessary context and commands to utilize the Iris CLI.

```bash
# Example: View the skill content
cat cli/SKILL.md
```
